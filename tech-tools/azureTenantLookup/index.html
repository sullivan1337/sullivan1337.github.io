Azure Tenant Lookup

Where's My Tenant?

Use this tool to query Azure AD for basic tenant information. This will show you:

  • If the tenant exists in Azure AD
  • What the tenant's GUID is
  • Which Azure AD instance the tenant resides in

To search, simply enter the tenant name, domain, email address, or the tenant's GUID in the search box below.

The tool will automatically search all Azure AD instances (Worldwide, US Government, China)

SAML HAR Analyzer

ℹ️ How to use

This tool analyzes SAML responses from HAR (HTTP Archive) files. All processing is done locally in your browser for privacy.

How to save a HAR file:

  1. Open your browser's Developer Tools (F12 or Ctrl+Shift+I)
  2. Go to the Network tab
  3. Perform the action that triggers the SAML request
  4. Right-click in the Network tab and select "Save all as HAR"

Then, upload the saved HAR file using the input below.

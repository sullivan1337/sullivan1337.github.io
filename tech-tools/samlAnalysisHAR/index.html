SAML HAR Analyzer

ℹ️ How to use

This tool analyzes SAML responses from HAR files or SAML tracer exports. All processing is done locally in your browser for privacy.

How to save a HAR file:

  1. Open your browser's Developer Tools (F12 or Ctrl+Shift+I)
  2. Go to the Network tab and enable "Preserve log"
    • Depending on browser and app configuration, you might need to enable "Auto-open DevTools for popups" in the "Global" DevTools Settings
  3. Perform the action that triggers the SAML request
  4. Right-click on the network output area to "Save all as HAR with content" or click the "Export HAR" download icon to save the HAR file
  5. Upload the saved HAR file or SAML tracer export file here to analyze the results
